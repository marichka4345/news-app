# News App

Gives opportunity to look through news of world's news sources
provided by News API (https://newsapi.org)

## Getting Started

To start webpack-dev-server in watch mode run:

```
npm run start
```

This will run the application on port 8080.



To create production bundle run:

```
npm run build
```
